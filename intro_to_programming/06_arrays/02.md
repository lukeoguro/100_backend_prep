# Exercise 2

```ruby
1. arr = ["b", "a"]
   arr = arr.product(Array(1..3))
   arr.first.delete(arr.first.last)

2. arr = ["b", "a"]
   arr = arr.product([Array(1..3)])
   arr.first.delete(arr.first.last)
```

1. Returns `1`. `arr` equals `[["b"], ["b", 2], ["b", 3], ["a", 1], ["a", 2], ["a", 3]]`

2. Returns `[1, 2, 3]`. `arr` equals `[["b", ["a", [1, 2, 3]]]`
